# Memory Palace Technique (Method of Loci)

A mnemonic strategy for memorising ordered lists by placing items along a mental route through a familiar building.

## How To Build a Memory Palace

1. Choose a familiar location (your house, office, commute route)
2. Identify 10-20 distinct stations along a consistent path
3. Create vivid, exaggerated mental images at each station
4. Walk the route mentally to encode, then again to retrieve

## Effectiveness

- Competition memory athletes use this as their primary technique
- Works best for ordered sequences and lists
- Less effective for conceptual understanding or procedural knowledge
- Requires significant practice to build and maintain palaces

## Limitations

- Not a substitute for understanding — it's rote memorisation
- Interference between palaces when using the same locations
- Does not leverage spaced repetition or prediction error signals
- Modern research suggests retrieval practice is more efficient for learning
