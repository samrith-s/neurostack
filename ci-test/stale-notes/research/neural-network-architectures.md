# Neural Network Architectures

Overview of modern deep learning architecture families for production ML systems.

## Transformer Family

- **GPT**: Decoder-only autoregressive models for text generation
- **BERT**: Encoder-only models for classification and NER tasks
- **T5**: Encoder-decoder for seq2seq tasks (translation, summarization)
- **Vision Transformer (ViT)**: Patches images into token sequences

## Convolutional Networks

- ResNet skip connections solve vanishing gradients in deep CNNs
- EfficientNet compound scaling balances depth, width, and resolution
- U-Net for semantic segmentation with skip connections

## Recurrent Networks (Legacy)

- LSTM gates control information flow through cell state
- GRU simplified gating mechanism, comparable performance to LSTM
- Largely replaced by transformers for sequence tasks since 2020
