# Prediction Errors in Learning

Prediction errors — the difference between what the brain expects and what it observes — are the fundamental signal driving learning and memory updating.

## The Predictive Brain

The brain constantly generates predictions about incoming information. When predictions fail:
- Dopaminergic neurons in the VTA/substantia nigra fire (reward prediction errors)
- Hierarchical prediction chains update their weights (perceptual prediction errors)
- Attention is directed to the surprising stimulus

## Types of Prediction Error

| Type | Signal | Effect |
|------|--------|--------|
| Positive surprise | Better than expected | Reinforces behaviour, strengthens associations |
| Negative surprise | Worse than expected | Weakens associations, triggers re-evaluation |
| Novelty | No prediction available | Triggers hippocampal encoding, exploration |

## Applications to Knowledge Management

In a knowledge vault, prediction errors surface when:
- A note appears in search results for an unexpected query (semantic drift)
- A note's content no longer matches its tags or folder context
- Two notes that should be linked are semantically distant

These signals indicate the vault needs maintenance — just as the brain updates its model when predictions fail. See [[spaced-repetition]] for how review intervals exploit prediction errors.
